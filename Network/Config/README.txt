To get info about what each script does, run the following:
  ./<FILENAME>.sh -h
  
OR keep reading for a brief description of each file.
create_AP_IP.sh - Sets up wlan1 as AP for devices to connect to.
  It also sets up the config scripts for the mesh network. Run this
  after hsmm-pi is installed.
  
hsmm-pi_copier - copys all hsmm-pi modified hsmm-pi files in the home 
  directory and copies them to the root directory hsmm-pi.
